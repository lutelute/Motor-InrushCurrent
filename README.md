# ⚡ モーター突入電流シミュレーター

モーター始動時の突入電流現象を視覚的に理解するためのインタラクティブシミュレーターです。

## 🚀 ライブデモ

**📱 すぐに試す:** [モーター突入電流シミュレーター](https://lutelute.github.io/Motor-InrushCurrent/)

## 📚 概要

このシミュレーターは、各種モーターの始動時に発生する突入電流の特性を詳細に解析・可視化します。投入位相角、DC成分、減衰特性など、突入電流の複雑な現象を直感的に理解できます。

### 🎯 学習目標

- **突入電流の物理現象**: なぜ始動時に大電流が流れるのか
- **DC成分の影響**: 投入位相角による非対称現象
- **モーター種類による差**: 各種モーターの始動特性比較
- **減衰特性**: 時定数と電流波形の関係

## 🔧 対象モーター

### 誘導機
1. **かご型誘導機** (Squirrel Cage)
   - 突入倍率: 6倍
   - 最も普及している構造
   - 特徴: 構造簡単、始動電流大

2. **巻線型誘導機** (Wound Rotor)  
   - 突入倍率: 3倍
   - 外部抵抗で電流制御可能
   - 特徴: 始動特性良好、構造複雑

### 同期機
3. **突極型同期機** (Salient Pole)
   - 突入倍率: 5倍
   - 低速・多極用途向き
   - 特徴: ダンパ巻線で始動

4. **円筒型同期機** (Cylindrical Rotor)
   - 突入倍率: 5倍  
   - 高速回転向き
   - 特徴: タービン直結型

## 📖 理論背景

### 💡 突入電流の発生メカニズム

1. **磁束鎖交数の急変**
   - 始動時の磁束変化
   - ファラデーの法則による誘起電圧

2. **過渡現象**
   - RL回路の過渡応答
   - DC成分の指数減衰

3. **非線形性**
   - 磁気飽和の影響
   - 高調波成分

### ⚡ DC成分の物理的意味

```
i(t) = I_ss·sin(ωt + φ) + I_dc·e^(-t/τ)
```

- **I_ss**: 定常状態電流振幅
- **φ**: 投入位相角
- **I_dc**: DC成分初期値  
- **τ**: 時定数

### 🔄 等価回路

#### 誘導機
```
V = I·(R₁ + jX₁ + (jX_m || (R₂'/s + jX₂')))
```

#### 同期機  
```
V = I·(R_a + jX_s) + E
```
